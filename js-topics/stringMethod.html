String Methods

In JavaScript, strings come with a rich set of methods that allow you to manipulate and interact with them in various ways. Here are some of the most common methods:

1. Using `charAt()`

The `charAt()` method returns the character at the specified index (position). 

let str = "Hello";
console.log(str.charAt(0)); // Output: H
console.log(str.charAt(1)); // Output: e

2. Using `charCodeAt()`

The `charCodeAt()` method returns the Unicode of the character at the specified index. 

let str = "Hello";
console.log(str.charCodeAt(0)); // Output: 72

3. Using `concat()`

The `concat()` method joins two or more strings and returns a new joined string. 

let str1 = "Hello";
let str2 = "World";
console.log(str1.concat(" ", str2)); // Output: Hello World

4. Using `includes()`

The `includes()` method checks whether a string contains the specified string/characters. 

let str = "Hello World";
console.log(str.includes("World")); // Output: true

5. Using `endsWith()`

The `endsWith()` method checks whether a string ends with specified string/characters. 

let str = "Hello World";
console.log(str.endsWith("World")); // Output: true

6. Using `indexOf()`

The `indexOf()` method returns the index of the first occurrence of a specified value in a string. 

let str = "Hello World";
console.log(str.indexOf("World")); // Output: 6

7. Using `lastIndexOf()`

The `lastIndexOf()` method returns the index of the last occurrence of a specified value in a string. 

let str = "Hello World World";
console.log(str.lastIndexOf("World")); // Output: 12

8. Using `match()`

The `match()` method searches a string for a match against a regular expression, and returns the matches. 

let str = "Hello World";
console.log(str.match(/World/g)); // Output: ["World"]

9. Using `matchAll()`

The `matchAll()` method returns an iterator of all matches of a string against a regular expression. 

let str = "Hello World World";
const matches = str.matchAll(/World/g);
for (const match of matches) {
    console.log(match[0]); // Output: World
}

10. Using `normalize()`

The `normalize()` method returns the Unicode Normalization Form of the string. 

let str = "é";
console.log(str.normalize("NFD")); // Output: é

11. Using `padEnd()`

The `padEnd()` method pads the current string with another string (repeated, if needed) until the resulting string reaches the given length. 

let str = "Hello";
console.log(str.padEnd(10, ".")); // Output: Hello.....

12. Using `padStart()`

The `padStart()` method pads the current string with another string (repeated, if needed) until the resulting string reaches the given length. 

let str = "Hello";
console.log(str.padStart(10, ".")); // Output: .....Hello

13. Using `repeat()`

The `repeat()` method returns a new string with a specified number of copies of the string it was called on. 

let str = "Hello";
console.log(str.repeat(3)); // Output: HelloHelloHello

14. Using `replace()`

The `replace()` method searches a string for a specified value, or a regular expression, and returns a new string where the specified values are replaced. 

let str = "Hello World";
console.log(str.replace("World", "Everyone")); // Output: Hello Everyone

15. Using `replaceAll()`

The `replaceAll()` method returns a new string with all matches of a pattern replaced by a replacement. 

let str = "Hello World World";
console.log(str.replaceAll("World", "Everyone")); // Output: Hello Everyone Everyone

16. Using `search()`

The `search()` method searches a string for a specified value and returns the position of the match. 

let str = "Hello World";
console.log(str.search("World")); // Output: 6

17. Using `slice()`

The `slice()` method extracts a part of a string and returns a new string. 

let str = "Hello World";
console.log(str.slice(0, 5)); // Output: Hello

18. Using `split()`

The `split()` method splits a string into an array of substrings.

let str = "Hello World";
console.log(str.split(" ")); // Output: ["Hello", "World"]

19. Using `startsWith()`

The `startsWith()` method checks whether a string begins with the characters of a specified string. 

let str = "Hello World";
console.log(str.startsWith("Hello")); // Output: true

20. Using `substring()`

The `substring()` method returns the part of the string between two specified indices. 

let str = "Hello World";
console.log(str.substring(0, 5)); // Output: Hello

21. Using `toLowerCase()`

The `toLowerCase()` method returns the calling string value converted to lowercase. 

let str = "Hello World";
console.log(str.toLowerCase()); // Output: hello world

22. Using `toUpperCase()`

The `toUpperCase()` method returns the calling string value converted to uppercase. 

let str = "Hello World";
console.log(str.toUpperCase()); // Output: HELLO WORLD

23. Using `trim()`

The `trim()` method removes whitespace from both ends of a string.

let str = "   Hello World   ";
console.log(str.trim()); // Output: Hello World

24. Using `trimEnd()`

The `trimEnd()` method removes whitespace from the end of a string.

let str = "   Hello World   ";
console.log(str.trimEnd()); // Output: "   Hello World"

25. Using `trimStart()`

The `trimStart()` method removes whitespace from the beginning of a string. 

let str = "   Hello World   ";
console.log(str.trimStart()); // Output: "Hello World   "

26. Using `valueOf()`

The `valueOf()` method returns the primitive value of a String object. 

let str = new String("Hello");
console.log(str.valueOf()); // Output: Hello
