Loops


Loops in JavaScript are used to repeatedly execute a block of code as long as a specified condition is true. JavaScript supports several types of loops.

1. for Loop

The for loop is used to run a block of code a specified number of times:

for (let i = 0; i < 5; i++) {
    console.log(i);
}
// Outputs: 0 1 2 3 4

2. while Loop

The while loop runs a block of code as long as the specified condition is true:

let i = 0;
while (i < 5) {
    console.log(i);
    i++;
}
// Outputs: 0 1 2 3 4

3. do...while Loop

The do...while loop runs a block of code once, before checking if the condition is true, then it repeats the loop as long as the condition is true:

let i = 0;
do {
    console.log(i);
    i++;
} while (i < 5);
// Outputs: 0 1 2 3 4

4. for...in Loop

The for...in loop iterates over the enumerable properties of an object:

const person = {name: "John", age: 30, city: "New York"};
for (let key in person) {
    console.log(key + ": " + person[key]);
}
// Outputs: 
// name: John
// age: 30
// city: New York

5. for...of Loop

The for...of loop iterates over the values of an iterable object (like an array):

const array = [1, 2, 3, 4, 5];
for (let value of array) {
    console.log(value);
}
// Outputs: 1 2 3 4 5

6. break and continue

The break statement exits the loop immediately, and the continue statement skips the current iteration and proceeds to the next iteration:

for (let i = 0; i < 5; i++) {
    if (i === 3) {
        break; // Exits the loop when i is 3
    }
    console.log(i);
}
// Outputs: 0 1 2

for (let i = 0; i < 5; i++) {
    if (i === 3) {
        continue; // Skips the iteration when i is 3
    }
    console.log(i);
}
// Outputs: 0 1 2 4
