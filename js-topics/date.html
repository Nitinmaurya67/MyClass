Dates


JavaScript provides the Date object to work with dates and times. The Date object is used to create, manipulate, and format dates and times.

1. Creating Dates

Dates can be created using the Date constructor:

const now = new Date(); // Current date and time
const specificDate = new Date('2024-09-27'); // Specific date
const dateTime = new Date('2024-09-27T14:00:00'); // Specific date and time
const timestamp = new Date(1632740400000); // Date from timestamp

2. Getting Date and Time Components

The Date object provides methods to get various components of a date and time:

const now = new Date();
console.log(now.getFullYear()); // Outputs: 2024
console.log(now.getMonth()); // Outputs: 8 (September, as months are zero-indexed)
console.log(now.getDate()); // Outputs: 27
console.log(now.getHours()); // Outputs: 14
console.log(now.getMinutes()); // Outputs: 0
console.log(now.getSeconds()); // Outputs: 0
console.log(now.getMilliseconds()); // Outputs: 0
console.log(now.getDay()); // Outputs: 5 (Friday, as days are zero-indexed starting from Sunday)

3. Setting Date and Time Components

The Date object provides methods to set various components of a date and time:

const now = new Date();
now.setFullYear(2025);
now.setMonth(0); // January
now.setDate(1);
now.setHours(12);
now.setMinutes(30);
now.setSeconds(45);
console.log(now); // Outputs: 2025-01-01T07:00:45.000Z (depending on your time zone)

4. Formatting Dates

JavaScript provides methods to format dates as strings:

const now = new Date();
console.log(now.toDateString()); // Outputs: Fri Sep 27 2024
console.log(now.toTimeString()); // Outputs: 14:00:00 GMT+0530 (India Standard Time)
console.log(now.toLocaleDateString()); // Outputs: 27/09/2024 (depending on locale)
console.log(now.toLocaleTimeString()); // Outputs: 14:00:00 (depending on locale)
console.log(now.toISOString()); // Outputs: 2024-09-27T08:30:00.000Z

5. Date Arithmetic

JavaScript allows you to perform arithmetic operations on dates:

const now = new Date();
const tomorrow = new Date(now);
tomorrow.setDate(now.getDate() + 1);
console.log(tomorrow); // Outputs: 2024-09-28T14:00:00.000Z (depending on your time zone)

const nextWeek = new Date(now);
nextWeek.setDate(now.getDate() + 7);
console.log(nextWeek); // Outputs: 2024-10-04T14:00:00.000Z (depending on your time zone)
